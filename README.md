# DVLD

DVLD (Driving Vehicle License Database) is a comprehensive system for managing driving licenses and vehicle registrations. It helps track license applications, test appointments, and vehicle data efficiently.

## Features
- Manage driving license applications
- Schedule and track test appointments
- Store and manage vehicle information
- User management with different roles
- Reports and statistics

## Technologies
- C# Windows Forms
- SQL Server
- ADO.NET for data access

## How to Run
1. Open the solution `DVLD.sln` in Visual Studio.
2. Build the project (Ctrl+Shift+B).
3. Run the application (F5).
4. Make sure SQL Server is installed and the database is configured properly.

## Project Structure
- `DVLDDataAccessLayer/` : Handles database interactions
- `DVLDBusinessLayer/` : Contains business logic
- `DVLD PresentationLayer Layer/` : User interface layer

## Notes
- Ensure your SQL Server instance is running before launching the app.
- Some features require administrator privileges in the app.
